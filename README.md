## ◈ SETUP & CONFIGURATION

### 1. Clone the Repo

```bash
git clone https://github.com/justfortestingnothibghere/TeamDev-X-NearalHub.git
cd TeamDev-X-NearalHub
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Configure `config.py`

```python
BOT_TOKEN   = "YOUR_TELEGRAM_BOT_TOKEN"        # @BotFather
API_SECRET  = "YOUR_SECRET_KEY"                 # Internal API auth
ADMIN_IDS   = [YOUR_TELEGRAM_USER_ID]           # Admin user IDs

MONGO_URI   = "mongodb+srv://..."               # MongoDB Atlas URI
MONGO_DB    = "teamdev_neuralhub"               # DB name

DL_API_BASE = "https://your-aio-api.railway.app" # AIO Downloader API URL
DL_API_KEY  = "YOUR_AIO_API_KEY"               # AIO API key (if required)

BATBIN_URL  = "https://batbin.me/YOUR_PASTE_ID" # Paste with Groq API keys (one per line)
GROQ_MODEL  = "llama-3.3-70b-versatile"
```

> 💡 **Groq Keys**: Create a free paste at [batbin.me](https://batbin.me) with your Groq API keys (one per line). The bot auto-rotates through them with failover.

### 4. Run

```bash
chmod +x start.sh
./start.sh
```

> `start.sh` launches the **Groq FastAPI service** → waits for readiness → starts **bot.py**
